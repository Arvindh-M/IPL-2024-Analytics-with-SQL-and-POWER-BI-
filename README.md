# IPL 2024 Data Analysis using SQL Server & Power BI

## Project Overview
This project analyzes IPL 2024 cricket data to derive insights on
team performance, player performance, and venue characteristics.
SQL Server was used for data storage and querying, while Power BI
was used for interactive dashboard visualization.

## Objectives
- Analyze team-wise batting and bowling performance
- Identify top-performing batters and bowlers
- Compare venue-wise scoring and bowling patterns
- Build interactive dashboards for data-driven insights

## Tools & Technologies
- SQL Server (Data cleaning & analysis)
- Power BI (Dashboarding & visualization)
- GitHub (Project version control)

## Dashboards Created
1. Team Performance Dashboard
2. Player Performance Dashboard
3. Venue Analysis Dashboard

## Key Insights
- Certain teams consistently score higher total runs
- A small set of players dominate run scoring and wicket-taking
- Some venues favor batting while others are bowler-friendly

## Repository Structure
- sql/ : SQL scripts for table creation and analysis
- powerbi/ : Power BI dashboard files
- dashboards/ : Dashboard screenshots
- data/ : Dataset description
